Описание решения задачи.
Условие задачи:
Написать программу, которая из имеющегося массива строк формирует массив из строк, длина которых
меньше либо равна 3 символа. Первоначальный массив можно ввести с клавиатуры, либо задать на старте 
выполнения алгоритма. При решении не рекомендуется пользоваться коллекциями, лучше обойтись исключительно 
массивами.

Задача будет реализована в виде управляющей программы и метода на языке программирования C# 
в среде .Net Framework 6.0 как консольное приложение.


Для решения задачи нужно:
1. В управляющей программе объявить строковую переменную (в качестве буферной для записи значений в массив строк),
переменную-массив типа string (для хранения введенных строк) и две переменных счетчика для обхода входного и
выходного массивов соответственно. Результат выполнения программы будет находиться в выходном массиве. Ввод
исходного массива строк в программе выполняется с клавиатуры. Ввод происходит в цикле и прекращается, как 
только пользователь вводит в массив строку "***"

2. В процессе работы программа вызывает пользовательский метод, который получает входной массив и
просматривает последовательно все его элементы, проверяя длину исходных строк.

3.Также данный метод создаёт изначально пустой выходной массив, в который будут записаны строки из входного массива,, длина которых
не превышает 3 символа. 

4. Как только во входном массиве найдена строка, удовлетворяющая условию, размер выходного массива
увеличивается на 1 с помощью метода Array.Resize() и данная строка записывается в качестве значения
добавленного элемента в выходной массив.

5. На выходе из цикла формируется выходной массив, который метод возвращает в управляющую программу.

6. Управляющая программа в цикле выводит все элементы выходного массива в консоль.
